# swno

shortcut usage of [swcr](https://www.npmjs.com/package/swcr), typescript node runtime powered by swc

## Install

```bash
  pnpm add -D swno
```

## Usage

```bash
  swno file.ts
```

## Inspiration

- [esno](https://github.com/antfu/esno) - TypeScript / ESNext node runtime powered by esbuild
