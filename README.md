# swno

Typescript simple runtime powered by Swc

## Intro

### [swcr](./packages/swcr)

Swc register for node.js. Efficient typescript runtime

#### Install

```bash
  pnpm add -D swcr @swc/core
```

#### Usage

```bash
  node -r swcr file.ts
```

### [swno](./packages/swno)

Typescript node runtime powered by swc

#### Install

```bash
  pnpm add -D swno
```

#### Usage

```bash
  swno file.ts
```
